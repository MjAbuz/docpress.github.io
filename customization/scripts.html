Custom Scripts

To specify custom scripts to be added to the page, you can use the scripts configuration option. You can use external script urls (hosted online) or local JS files (local files need to be inside the docs/ directory).

docpress.json
{
  "scripts": [
    "http://site.com/external.js",
    "scripts/myscript.js"
  ]
}

Scripts will be included in the <head> tag, after the base theme assets/script.js file.
